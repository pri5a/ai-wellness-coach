Overview:
AI Wellness Coach is a Salesforce-based application designed to help users track daily wellness data and receive personalized feedback.
The app allows users to log key health metrics such as sleep, workout, hydration, mood. Based on this data, the system analyzes trends over time and generates coaching insights that help users improve their overall health and consistency.
This project is built using Lightning Web Components, Apex Services, integration, and scalable architecture patterns.

Key Features:
- Daily wellness logging using Lightning Web Components.
- Apex services for data processing.
- Trend analysis across multiple health metrics.
- Rule-based AI-style coaching insights.
- Modular architecture designed for future AI integrations.

Technical skills:
- Salesforce Lightning Web Components (LWC)
- Apex
- Custom Objects and Fields
- SOQL and Data Aggregation
- Git and GitHub for version control
